D — Shell Is the Platform, Canvas Is an Extension

The shell hosts any canvas type. Each extension brings its own. Multiple canvas types coexist via tabs. The shell itself stays canvas-agnostic.

You can pick more than one if your vision combines types — e.g. D + A, where the shell is extensible but ships with a spatial canvas built-in.
