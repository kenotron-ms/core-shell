Data flow — end to end example

2
Main process — chokidar fires
Sends ipc:file-changed event to renderer with path + new bytes
3
Extension callback fires
shell.ipc.watchDir callback in amplifier-canvas receives the event
4
Extension store updates
setState({ sessions: parsedSessions }) — Zustand notifies subscribers
5
React re-renders
