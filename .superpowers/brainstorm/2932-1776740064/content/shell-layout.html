Extension contribution summary

activityRail
Icon + associated sidebar view. Clicking activates the sidebar panel.
sidebar
Left panel content. Can be any React component — session list, file tree, anything.
canvas ★
The main content area. This is the extension's primary UI. One active at a time.
rightSidebar
Optional properties/context panel. Shell or extension can contribute. Collapsible.
panel
Bottom panel tab. Terminal, output, logs — multiple tabs, multiple contributors.
statusBar
Left or right-aligned status items. Small — icons + short text only.
